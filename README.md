GodFall
    GodFall is a strategy board game played on an 8x8 grid.
    It is inspired by chess but has its own rules and pieces.
        
        The Two Sides-
            You play as Olympus or Underworld.
            Each side has the same set of pieces but with different names.
        
        The Pieces-
            Zeus, the king. You lose if he is captured.
            Athena, jumps in an extended knight pattern.
            Hermes, moves up to three squares in a straight line.
            Ares, must attack nearby enemies if any are in range.
            Artemis, moves diagonally and can capture from two squares away.
            Hephaestus, cannot move but has three hit points.
            Apollo, moves one step in any direction and can swap with a friendly piece.
            Soldier, moves forward and captures diagonally. Promotes on the last row.

        How to Win-
            Capture the enemy Zeus to win.
            This is called a Godfall.

        Playing Against the AI-
            The game has a built in AI opponent.
            It looks a few moves ahead to choose the best move it can find.
            
        Running the Game-
            Run main.py to start the game.
